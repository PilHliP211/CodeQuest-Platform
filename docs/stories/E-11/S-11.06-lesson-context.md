## What

`LessonContext` wraps the `lessonReducer` in a React context so any component in the lesson screen tree can read lesson state and dispatch actions without prop drilling. It follows the exact same pattern as `ProfileContext` from E-02.

## Tasks

1. Create `src/engine/LessonContext.tsx`:
   ```tsx
   import { createContext, useContext, useReducer, useMemo, type ReactNode } from 'react';
   import type { Lesson } from '@/types/content';
   import type { LessonState, LessonAction } from '@/types/lessonState';
   import { lessonReducer, createInitialState } from './lessonRunner';

   interface LessonContextValue {
     lessonState: LessonState;
     dispatch: React.Dispatch<LessonAction>;
     lesson: Lesson;
   }

   const LessonContext = createContext<LessonContextValue | undefined>(undefined);

   export function LessonProvider({
     lesson,
     children,
   }: {
     lesson: Lesson;
     children: ReactNode;
   }): React.ReactElement {
     const [lessonState, dispatch] = useReducer(
       lessonReducer,
       lesson.id,
       createInitialState,
     );

     const value = useMemo<LessonContextValue>(
       () => ({ lessonState, dispatch, lesson }),
       [lessonState, lesson],
     );

     return <LessonContext.Provider value={value}>{children}</LessonContext.Provider>;
   }

   export function useLessonRunner(): LessonContextValue {
     const ctx = useContext(LessonContext);
     if (ctx === undefined) {
       throw new Error('useLessonRunner must be used inside <LessonProvider>');
     }
     return ctx;
   }
   ```

2. The `LessonProvider` is rendered by `LessonScreen.tsx` (the top-level lesson view, built later). It receives the active `lesson` object from `useContent()`.

3. Export `LessonProvider` and `useLessonRunner` from `src/engine/LessonContext.tsx`.

4. Create a stub `LessonScreen.tsx` in `src/components/Lesson/`:
   ```tsx
   import { LessonProvider } from '@/engine/LessonContext';
   import type { Lesson } from '@/types/content';

   export function LessonScreen({ lesson }: { lesson: Lesson }): React.ReactElement {
     return (
       <LessonProvider lesson={lesson}>
         {/* Phase UI components go here in E-11 / future epics */}
         <div className="font-pixel">Lesson: {lesson.title}</div>
       </LessonProvider>
     );
   }
   ```

5. In `App.tsx` (or `MapScreen.tsx`), wire up: when a node is selected on the map (`onNodeSelect`), find the corresponding `Lesson` from `useContent()` and render `<LessonScreen lesson={...} />`.
