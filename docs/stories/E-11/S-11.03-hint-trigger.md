## What

The lesson JSON defines a `hints` array per phase challenge: `[{ afterAttempts: 2, text: "..." }, { afterAttempts: 4, text: "..." }]`. When `attemptCount` reaches `hint.afterAttempts`, that hint becomes active. This story adds a selector function and documents how the hint UI is triggered.

The hint text is never stored in `LessonState` тАФ it's derived on each render from `attemptCount` + the lesson's hint array. This avoids redundant state.

## Tasks

1. Create `src/engine/hintSelector.ts`:
   ```ts
   import type { Hint } from '@/types/content';

   /**
    * Returns the most relevant hint for the current attempt count,
    * or null if no hint threshold has been reached yet.
    */
   export function getActiveHint(hints: Hint[], attemptCount: number): string | null {
     // Find the highest threshold that has been reached
     const active = hints
       .filter((h) => attemptCount >= h.afterAttempts)
       .sort((a, b) => b.afterAttempts - a.afterAttempts)[0];
     return active?.text ?? null;
   }
   ```

2. The `Hint` type is `{ afterAttempts: number; text: string }` тАФ already defined in `src/types/content.ts` (E-03).

3. In the lesson screen component (E-11, S-11.06), call this selector:
   ```tsx
   const currentPhaseHints = getCurrentPhaseHints(lesson, lessonState.currentPhase);
   const activeHint = getActiveHint(currentPhaseHints, lessonState.attemptCount);
   ```

4. The hint UI renders as a collapsible "Hint ЁЯТб" button that expands to show `activeHint`. When `activeHint` is `null`, the button is hidden (not just disabled). When it appears, it should attract attention without being disruptive.

5. `SHOW_HINT` action (S-11.01) is dispatched when the player opens the hint. This increments `hintsUsed` in the state, which is later used for XP penalty calculation (S-11.05).

6. A helper to get the current phase's hint array:
   ```ts
   function getCurrentPhaseHints(lesson: Lesson, phase: LessonPhase): Hint[] {
     if (phase === 1) return lesson.phase1.challenge.hints ?? [];
     if (phase === 2) return lesson.phase2.challenge.hints ?? [];
     if (phase === 3) return lesson.phase3.hints ?? [];
     return [];
   }
   ```
