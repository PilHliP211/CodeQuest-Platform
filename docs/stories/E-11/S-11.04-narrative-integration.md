## What

When the lesson runner transitions into a narrative-playing state (`phaseStatus: 'narrative-intro'` or `'narrative-outro'`), it needs to know *which* script to show. This story populates `pendingNarrative` in `LessonState` at the right transitions.

The lesson JSON contains narrative scripts at defined locations. The lesson runner (or the context layer above it) looks up the appropriate script and sets `pendingNarrative` when entering a narrative phase status.

## Tasks

1. Extend `LessonContext` (S-11.06) to accept the active `lesson: Lesson` object and a helper to look up the current narrative script:
   ```ts
   function getPendingNarrative(lesson: Lesson, state: LessonState): NarrativeScene[] | null {
     const { currentPhase, phaseStatus } = state;
     if (currentPhase === 1 && phaseStatus === 'narrative-intro') return lesson.phase1.intro.scenes;
     if (currentPhase === 1 && phaseStatus === 'narrative-outro') return lesson.phase1.outro.scenes;
     if (currentPhase === 2 && phaseStatus === 'narrative-intro') return lesson.phase2.intro.scenes;
     if (currentPhase === 2 && phaseStatus === 'narrative-outro') return lesson.phase2.reveal.scenes;
     if (currentPhase === 3 && phaseStatus === 'narrative-intro') return lesson.phase3.intro.scenes;
     if (currentPhase === 'complete') return null; // celebration handled separately
     return null;
   }
   ```

2. In the lesson screen, when `phaseStatus` is `'narrative-intro'` or `'narrative-outro'`, render `<NarrativePlayer>` with the current script. When `NarrativePlayer` calls `onComplete`, dispatch `NARRATIVE_COMPLETE` to the lesson runner.

3. The celebration scene (Phase 3 complete) is handled separately: when `currentPhase === 'complete'`, show the `phase3.celebration` scene (a narrative with the flag reveal animation). After the celebration narrative, dispatch `COMPLETE_LESSON` if not already done, then navigate back to the map.

4. The `NarrativePlayer` is always `skippable={true}` except for the very first intro of a lesson (first-time play). For replays, the player has already seen the story and should be able to skip freely.
