# S-14.03: Phase 1 Narrative Scripts

**Epic:** E-14
**Size:** S
**Dependencies:** S-14.02 (lesson.json scaffold)

## Skills to Load

- `testing-strategy` — background on the outcome-focused validation philosophy; this story adds no executable code, so no new tests are required
- `content-pack-system` — narrative script format: `scenes[].dialogue[].speaker/text/portrait`

## What

Write the full dialogue for Phase 1 `intro` and `outro` scenes in `lesson.json`. The tone is adventurous and age-appropriate for kids 8–12: short sentences, active voice, mild suspense. The villain is mysterious (name unknown — "??"). The player is a young coder with an agency that fights against the villain's history-erasing scheme.

Phase 1 concept: **sequences** — doing things in order.
