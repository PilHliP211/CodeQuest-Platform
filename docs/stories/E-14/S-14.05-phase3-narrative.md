# S-14.05: Phase 3 Narrative Scripts + Celebration

**Epic:** E-14
**Size:** S
**Dependencies:** S-14.02 (lesson.json scaffold)

## Skills to Load

- `content-pack-system` — `phase3.intro.scenes` and `phase3.celebration`

## What

Write Phase 3 `intro` dialogue and the `celebration` config. Phase 3 is where the player writes real code (in the syntax editor) to rebuild Japan's flag on the canvas. The intro should explain what the canvas is and set up the challenge clearly.

The celebration is triggered when the player's canvas output matches the solution. It shows a final message and the flag-restore animation.

## Tasks

**Phase 3 `intro` (2 scenes):**
```json
"intro": {
  "scenes": [
    {
      "background": "assets/phase3-bg.png",
      "dialogue": [
        {
          "speaker": "Hana",
          "text": "This is the Restoration Canvas. It's blank — but you have everything you need to rebuild the flag.",
          "portrait": "assets/guide-portrait.png"
        },
        {
          "speaker": "Hana",
          "text": "You've got two functions: fillBackground to set the color, and drawCircle to draw the red circle. Use them in order.",
          "portrait": "assets/guide-portrait.png"
        }
      ],
      "advanceOnInput": true
    },
    {
      "background": "assets/phase3-bg.png",
      "dialogue": [
        {
          "speaker": "Player",
          "text": "fillBackground... then drawCircle. I've got this. The syntax editor is unlocked — time to write some real code.",
          "portrait": "assets/player-portrait.png"
        }
      ],
      "advanceOnInput": true
    }
  ]
}
```

**`celebration` config:**
```json
"celebration": {
  "animation": "flag-restore",
  "message": "Japan's flag has been restored! The villain's plan has failed — for now. More countries need your help.",
  "xpBonus": 50
}
```

Note: The `celebration` message should be shown in a narrative scene after the flag-restore animation plays. The lesson runner (E-11) handles triggering this scene on `COMPLETE_LESSON`.

## Done When

- [ ] Phase 3 `intro` has 2 scenes
- [ ] Both available functions mentioned naturally in dialogue
- [ ] `celebration.message` written with a hook toward future lessons ("More countries need your help")
- [ ] `celebration.animation` set to `"flag-restore"`
- [ ] `celebration.xpBonus` set to `50`
- [ ] JSON valid
