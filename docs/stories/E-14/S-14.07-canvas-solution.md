# S-14.07: Canvas Solution Definition

**Epic:** E-14
**Size:** XS
**Dependencies:** S-14.02 (lesson.json scaffold — `phase3.canvas`)

## Skills to Load

- `testing-strategy` — background on the outcome-focused validation philosophy; this story adds no executable code, so no new tests are required
- `content-pack-system` — `phase3.canvas.solution` and `tolerance` fields

## What

Verify the canvas solution definition is complete and correct. This is already scaffolded in S-14.02 — this story double-checks the values and documents why they're correct.

## Tasks

1. Confirm `phase3.canvas.solution` in `lesson.json`:
   ```json
   "solution": {
     "calls": [
       { "fn": "fillBackground", "args": ["white"] },
       { "fn": "drawCircle", "args": [150, 100, 60, "red"] }
     ]
   }
   ```

2. Confirm `"tolerance": 15` — this means the player's circle center can be up to 15px off from (150, 100) and the radius can be up to 15px off from 60 and it still counts as correct. This is generous enough for kids to succeed without pixel-perfect precision.

3. Confirm `availableFunctions` includes only `fillBackground` and `drawCircle`. No other drawing functions are available in this lesson.

4. Manually verify the solution makes the Japan flag:
   - `fillBackground("white")` → entire 300×200 canvas becomes white
   - `drawCircle(150, 100, 60, "red")` → red circle centered at (150, 100) = center of canvas, radius 60

   On a 300×200 canvas, center is (150, 100). The circle occupies roughly 40% of the canvas width — correct proportions for the Japanese flag (the hinomaru circle is roughly 3/8 of the flag width).

5. Document: there is no scoring for *how* close to perfect the drawing is — `evaluateSuccess` returns `true`/`false`. A circle at (145, 95, 62) would pass; a circle at (200, 150, 60) would fail (x is 50px off, exceeding tolerance of 15).

## Done When

- [ ] `phase3.canvas.solution.calls` has exactly 2 calls
- [ ] `tolerance: 15` set
- [ ] `availableFunctions` has exactly `fillBackground` and `drawCircle`
- [ ] Canvas dimensions are 300×200
- [ ] Manually verified that the solution calls produce the Japan flag
