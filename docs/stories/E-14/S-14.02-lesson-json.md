# S-14.02: `lesson.json` Scaffold (Japan, All 3 Phases)

**Epic:** E-14
**Size:** M
**Dependencies:** S-14.01 (course.json), S-03.01 (Lesson type for schema reference)

## Skills to Load

- `content-pack-system` — `lesson.json` schema per CONTENT_SCHEMA.md

## What

Create the full `lesson.json` for the Japan lesson. This is the largest content file — it defines all 3 phases, all block definitions, and the canvas challenge. Narrative scripts are left as short placeholders in this story; they're expanded in S-14.03 through S-14.05.
