## What

Write Phase 2 `intro` and `reveal` dialogue. Introduce **Hana** — a local guide who helps the player track down the flag's missing pieces. Phase 2 is about the "Investigate" phase: searching for and collecting the flag ingredients (white background, red circle) using blocks that now show their code syntax.

The dialogue should mention the blocks' code labels in a natural way — e.g. "Those blocks are showing you the actual commands they run. Pretty cool, right?"
