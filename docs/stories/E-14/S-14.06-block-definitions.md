# S-14.06: Block Definitions (All 8 Blocks)

**Epic:** E-14
**Size:** S
**Dependencies:** S-14.02 (lesson.json `blocks` array)

## Skills to Load

- `content-pack-system` — block definition format per CONTENT_SCHEMA.md

## What

Write all 8 block definitions in `lesson.json`'s `blocks` array. These are the custom Blockly block types used across all 3 phases. Block colors follow the category table in CONTENT_SCHEMA.md (Movement=160, Search/Collect=210, Drawing=20).
