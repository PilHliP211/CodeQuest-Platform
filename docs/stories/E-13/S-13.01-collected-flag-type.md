# S-13.01: `CollectedFlag` and `CollectionState` Types

**Epic:** E-13
**Size:** XS
**Dependencies:** S-12.01 (progress types), S-03.01 (content types — `MapNode`)

## Skills to Load

- `typescript-standards` — interface definitions, no `any`
- `naming-conventions` — extend `src/types/progress.ts` or add `src/types/collection.ts`

## What

Define the types needed by the flag collection UI. A `CollectedFlag` entry combines the display data (label, icon path) with the metadata (when it was collected). It's derived from the `LessonProgress.flagCollected` flag + the `MapNode` definition in the content pack.

These types can live in `src/types/progress.ts` (extending the existing progress types file) or a dedicated `src/types/collection.ts` — either is acceptable. Prefer extending the progress file to avoid fragmentation.

## Tasks

1. Add to `src/types/progress.ts`:
   ```ts
   /** A collected flag entry — derived from LessonProgress + MapNode */
   export interface CollectedFlag {
     lessonId: string;
     label: string;           // MapNode.label (e.g. "Japan")
     iconPath: string;        // MapNode.collectedIcon (relative path)
     collectedAt: string;     // ISO 8601 timestamp from LessonProgress.completedAt
   }
   ```

2. No store type is needed — the collection state is derived on each render from `progressStore.getLessonProgress()` + `course.map.nodes`. There's no need to persist a separate `CollectedFlag[]` array; it's always computable.

3. Export a helper to build the flag list:
   ```ts
   // In src/engine/collectionDeriver.ts
   import type { MapNode } from '@/types/content';
   import type { CollectedFlag } from '@/types/progress';
   import { getLessonProgress } from './progressStore';

   export function deriveCollectedFlags(nodes: MapNode[], packId: string): CollectedFlag[] {
     return nodes
       .map((node): CollectedFlag | null => {
         const progress = getLessonProgress(packId, node.id);
         if (!progress?.flagCollected) return null;
         return {
           lessonId: node.id,
           label: node.label,
           iconPath: node.collectedIcon,
           collectedAt: progress.completedAt ?? new Date().toISOString(),
         };
       })
       .filter((f): f is CollectedFlag => f !== null);
   }
   ```
