# S-13.02: `FlagTile.tsx` — Single Tile, Collected vs. Uncollected States

**Epic:** E-13
**Size:** S
**Dependencies:** S-13.01 (CollectedFlag type), S-03.01 (MapNode type)

## Skills to Load

- `react-component-structure` — named export, props interface
- `accessibility` — `role="img"` with descriptive `aria-label`, focus handling
- `formatting` — CSS `image-rendering: pixelated` for flag images

## What

`FlagTile` renders a single flag slot. When collected: full-color flag image, country name label, and completion date visible on hover/focus. When uncollected: silhouette (grayscale CSS filter) with "???" label. A pixel-art frame/border surrounds the tile in both states.

## Tasks

1. Create `src/components/Collection/FlagTile.tsx`:
   ```tsx
   import type { MapNode } from '@/types/content';
   import type { CollectedFlag } from '@/types/progress';

   interface FlagTileProps {
     node: MapNode;
     collected: CollectedFlag | null;
   }

   export function FlagTile({ node, collected }: FlagTileProps): React.ReactElement {
     const isCollected = collected !== null;
     const label = isCollected ? node.label : '???';
     const ariaLabel = isCollected
       ? `${node.label} flag — collected on ${new Date(collected.collectedAt).toLocaleDateString()}`
       : `Unknown flag — not yet collected`;

     return (
       <div
         className="relative border-4 border-gray-400 bg-gray-800 p-2 w-24 h-16 flex flex-col items-center justify-center group focus-within:border-yellow-400"
         role="img"
         aria-label={ariaLabel}
       >
         <img
           src={isCollected ? node.collectedIcon : node.collectedIcon}
           alt=""
           aria-hidden="true"
           className={`w-full h-full object-contain ${isCollected ? '' : 'grayscale opacity-30'}`}
           style={{ imageRendering: 'pixelated' }}
         />
         <span className="font-pixel text-xs text-center mt-1 text-white truncate w-full text-center">
           {label}
         </span>
         {isCollected && (
           <div className="absolute inset-0 bg-black bg-opacity-80 hidden group-hover:flex group-focus-within:flex items-center justify-center text-center p-1">
             <span className="font-pixel text-xs text-yellow-300">
               {node.label}{'\n'}
               {new Date(collected.collectedAt).toLocaleDateString()}
             </span>
           </div>
         )}
       </div>
     );
   }
   ```

2. The silhouette is achieved via `grayscale opacity-30` CSS classes (Tailwind). Do not use a separate silhouette image — use CSS filter on the same icon.

3. The hover tooltip (completion date) uses Tailwind `group-hover` — make sure the parent div has `group` class.

4. Pixel-art frame: `border-4 border-gray-400` gives a chunky pixel-art border. On keyboard focus (`focus-within`), change to `border-yellow-400` to show keyboard navigation.
