# S-13.05: Collection Screen Navigation (Map → Collection → Map)

**Epic:** E-13
**Size:** XS
**Dependencies:** S-13.03 (CollectionGrid), S-04.08 (MapScreen)

## Skills to Load

- `react-component-structure` — conditional rendering pattern
- `accessibility` — overlay with focus trap, Escape key to close

## What

A "Collection" button on the map screen opens the `CollectionGrid` as a full-screen overlay. A "Back ←" button (or Escape key) returns to the map. This is the simplest possible navigation: a boolean `isOpen` state, no router.

## Tasks

1. Add collection state to `MapScreen.tsx`:
   ```tsx
   const [showCollection, setShowCollection] = useState(false);

   if (showCollection) {
     return (
       <div className="fixed inset-0 bg-gray-950 overflow-y-auto p-6 z-50">
         <button
           type="button"
           onClick={() => setShowCollection(false)}
           className="font-pixel text-sm text-white mb-4"
           autoFocus
         >
           ← Back to Map
         </button>
         <CollectionGrid />
       </div>
     );
   }
   ```

2. Add a "Collection 🏳" button to `MapScreen`'s main render (beside or below the map):
   ```tsx
   <button
     type="button"
     onClick={() => setShowCollection(true)}
     className="font-pixel text-sm"
   >
     Collection 🏳
   </button>
   ```

3. `Escape` key closes the collection overlay:
   ```ts
   useEffect(() => {
     function onKey(e: KeyboardEvent): void {
       if (e.key === 'Escape') setShowCollection(false);
     }
     if (showCollection) window.addEventListener('keydown', onKey);
     return () => window.removeEventListener('keydown', onKey);
   }, [showCollection]);
   ```

4. The "Back" button receives `autoFocus` so keyboard users land on it immediately when the overlay opens.

## Done When

- [ ] "Collection" button on map screen opens collection overlay
- [ ] "Back to Map" button closes the overlay
- [ ] `Escape` key closes the overlay
