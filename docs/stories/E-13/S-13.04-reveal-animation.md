# S-13.04: New Flag Reveal Animation

**Epic:** E-13
**Size:** S
**Dependencies:** S-13.02 (FlagTile), S-13.03 (CollectionGrid)

## Skills to Load

- `testing-strategy` — **required** — outcome-focused validation before writing any test
- `component-tests` — Testing Library behavior tests for the component added here
- `react-component-structure` — `useEffect` watching a derived value
- `formatting` — CSS keyframe animation for the flip/reveal

## What

When a flag is newly collected (the player just completed a lesson), the corresponding `FlagTile` should animate from silhouette to full color — a flip or scale-up effect that makes the moment feel rewarding. This is triggered by detecting that the collection count has grown since the last render.

The animation is CSS-driven (Tailwind + a custom keyframe) — no external animation library needed.

## Tasks

1. Add a `isNew` prop to `FlagTile`:
   ```tsx
   interface FlagTileProps {
     node: MapNode;
     collected: CollectedFlag | null;
     isNew?: boolean; // true if this flag was just collected this session
   }
   ```
   When `isNew` is true, apply an animation class:
   ```tsx
   className={`... ${isNew === true ? 'animate-flag-reveal' : ''}`}
   ```

2. Define the `animate-flag-reveal` keyframe in `src/index.css` (or `tailwind.config.ts` `extend.animation`):
   ```css
   @keyframes flag-reveal {
     0%   { transform: scale(0.5) rotateY(90deg); opacity: 0; }
     60%  { transform: scale(1.2) rotateY(0deg); opacity: 1; }
     100% { transform: scale(1) rotateY(0deg); opacity: 1; }
   }
   .animate-flag-reveal {
     animation: flag-reveal 0.6s ease-out forwards;
   }
   ```

3. In `CollectionGrid`, track the "previous collected count" with a `useRef`. When the count increases, find the newly collected flag ID(s) and pass `isNew={true}` to those tiles:
   ```tsx
   const prevCountRef = useRef(collectedFlags.length);
   const newFlagIds = new Set<string>();

   if (collectedFlags.length > prevCountRef.current) {
     // Find flags collected after prevCount
     const newlyCollected = collectedFlags.slice(prevCountRef.current);
     for (const f of newlyCollected) newFlagIds.add(f.lessonId);
   }
   prevCountRef.current = collectedFlags.length;
   ```

4. The animation only plays once per session — subsequent mounts of `CollectionGrid` will not replay it (the flag was already collected before this session's `prevCountRef` started). This is the correct behaviour; the animation is a "new this session" indicator, not "ever newly collected."
