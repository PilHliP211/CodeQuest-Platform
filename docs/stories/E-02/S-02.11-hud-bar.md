# S-02.11: HUD Bar Component

**Epic:** E-02
**Size:** XS
**Dependencies:** S-02.06

## Skills to Load

- `react-component-structure` — file template
- `react-context-pattern` — consuming `useProfile`

## What

A persistent top bar (HUD) showing the learner's name. Just the name for now — XP and other elements come in later epics.

## Tasks

1. Create `src/components/HUD/HUD.tsx`:
   ```tsx
   import { useProfile } from '@/engine/ProfileContext';

   export function HUD(): React.ReactElement {
     const { profile } = useProfile();
     return (
       <header className="font-pixel ...">
         <span>{profile?.name ?? ''}</span>
       </header>
     );
   }
   ```
2. Style as a fixed-height top bar (Tailwind). Doesn't need to be fancy — just visible and styled with the pixel font.
3. Note: the `?? ''` is defensive but the HUD should never render without a profile (the gate ensures this). It's there to satisfy the type system without an exclamation mark.

## Done When

- [x] HUD component renders the learner's name
- [x] Uses the pixel font
- [x] No `!` non-null assertions
- [x] `npm run lint` and `npm run typecheck` pass
