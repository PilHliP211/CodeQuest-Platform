# S-02.06: useProfile Hook

**Epic:** E-02
**Size:** XS
**Dependencies:** S-02.05

## Skills to Load

- `react-context-pattern` ‚Äî the hook-with-provider-check pattern

## What

The `useProfile` hook reads from `ProfileContext` and throws if used outside the provider. This is the only public consumer API for profile state.

## Tasks

1. Add to `src/engine/ProfileContext.tsx` (or extract to `src/engine/useProfile.ts` if you prefer):
   ```ts
   import { useContext } from 'react';
   import { ProfileContext, type ProfileContextValue } from './ProfileContext';

   export function useProfile(): ProfileContextValue {
     const ctx = useContext(ProfileContext);
     if (ctx === undefined) {
       throw new Error('useProfile must be used within <ProfileProvider>');
     }
     return ctx;
   }
   ```
2. Do NOT export the raw `ProfileContext` for direct `useContext` use ‚Äî the hook is the only entry point.

## Done When
