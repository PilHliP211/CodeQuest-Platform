# S-02.01: LearnerProfile Type

**Epic:** E-02
**Size:** XS
**Dependencies:** S-01.05 (folder structure)

## Skills to Load

- `typescript-standards` — interface vs type, no `any`
- `naming-conventions` — file naming for `src/types/`
- `architecture-overview` — confirms types live in `src/types/`

## What

Define the `LearnerProfile` interface. This is the shape that's stored in localStorage and consumed by every component that needs the learner's name.

## Tasks

1. Create `src/types/profile.ts`:
   ```ts
   export interface LearnerProfile {
     name: string;
     createdAt: string; // ISO 8601 timestamp
   }
   ```
2. That's it. No validation function yet (S-02.03), no store (S-02.04).
