# S-02.12: HUDLayout Wrapper

**Epic:** E-02
**Size:** XS
**Dependencies:** S-02.11

## Skills to Load

- `react-component-structure` — children prop pattern

## What

A layout component that renders the HUD bar at the top and the page content below it. This is the standard wrapper for all gameplay screens. It is NOT used for the name entry screen.

## Tasks

1. Create `src/components/HUD/HUDLayout.tsx`:
   ```tsx
   import type { ReactNode } from 'react';
   import { HUD } from './HUD';

   interface HUDLayoutProps {
     children: ReactNode;
   }

   export function HUDLayout({ children }: HUDLayoutProps): React.ReactElement {
     return (
       <div className="flex min-h-screen flex-col">
         <HUD />
         <main className="flex-1">{children}</main>
       </div>
     );
   }
   ```
2. Update `App.tsx` to wrap the main app branch in `<HUDLayout>`:
   ```tsx
   if (profile === null) return <NameEntryScreen />;
   return <HUDLayout>{/* main app */}</HUDLayout>;
   ```

## Done When

- [x] `HUDLayout` exists and wraps children below the HUD
- [x] Used in `App.tsx` for the post-profile branch
- [x] HUD does NOT show on the name entry screen
- [x] Layout fills the viewport vertically
