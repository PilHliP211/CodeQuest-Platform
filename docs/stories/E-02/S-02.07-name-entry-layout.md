# S-02.07: NameEntryScreen Layout & Styling

**Epic:** E-02
**Size:** S
**Dependencies:** S-02.06, S-01.13 (pixel font)

## Skills to Load

- `react-component-structure` — file template, named export, props pattern
- `naming-conventions` — `NameEntryScreen.tsx` in `src/components/Profile/`

## What

The visual structure of the first-launch name entry screen. Validation logic comes in S-02.08; accessibility in S-02.09. This story is layout + styling only.

## Tasks

1. Create `src/components/Profile/NameEntryScreen.tsx`:
   - Full-screen, centered layout (Tailwind)
   - Pixel-art heading: "What's your name, adventurer?"
   - Single text input
   - Submit button: "Start Quest!"
   - Pixel font on heading and button
2. The component takes no props for now — it reads `createProfile` from `useProfile()` directly.
3. On submit (no validation yet): call `createProfile(name)`.
4. Use named export, explicit return type, props interface co-located if any are added.
