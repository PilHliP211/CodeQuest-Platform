# S-04.08: MapScreen Component

**Epic:** E-04
**Size:** XS
**Dependencies:** S-04.05 (MapCanvas), S-04.03 (PlayerMarker), S-04.04 (VillainTrail), S-04.07 (animated transition), S-03.05 (useContent)

## Skills to Load

- `react-component-structure` — screen-level composition
- `content-pack-system` — reading player and villain sprite paths from `useContent`
- `typescript-standards` — typed props, no `any`

## What

`MapScreen.tsx` is the top-level screen component that `App.tsx` renders when no lesson is active. It composes `MapCanvas`, `PlayerMarker`, `VillainTrail`, and wires up the `onNodeSelect` callback.

`MapCanvas` owns node rendering and edge lines. `PlayerMarker` and `VillainTrail` are rendered by `MapScreen` on top of `MapCanvas` (both are absolutely positioned inside the same relative container that `MapCanvas` establishes — or `MapScreen` creates its own wrapper that contains all of them).

The simplest approach: `MapScreen` renders a single `position: relative` wrapper at the map's pixel dimensions, then renders `MapCanvas`, `PlayerMarker`, and `VillainTrail` as children (all `position: absolute`).

## Tasks

1. Create `src/components/Map/MapScreen.tsx`:
   ```tsx
   import { useState } from 'react';
   import { useContent } from '@/engine/ContentContext';
   import { MapCanvas } from './MapCanvas';
   import { PlayerMarker } from './PlayerMarker';
   import { VillainTrail } from './VillainTrail';

   export function MapScreen(): React.ReactElement {
     const { course } = useContent();
     const { map } = course;

     // Stub: current node comes from progress store (E-12)
     const [currentNodeId, setCurrentNodeId] = useState<string>(map.startNodeId);

     // Stub: animateTo comes from lesson runner (E-11)
     const animateTo: string | undefined = undefined;

     function handleNodeSelect(nodeId: string): void {
       // TODO(E-11): trigger lesson start for nodeId
       console.warn('Node selected — lesson start not yet implemented:', nodeId);
     }

     function handleAnimationEnd(): void {
       // After animation, update current node to the destination
       if (animateTo !== undefined) {
         setCurrentNodeId(animateTo);
       }
     }

     const currentNode = map.nodes.find((n) => n.id === currentNodeId);
     const currentPixelPosition =
       currentNode !== undefined
         ? { x: currentNode.x * map.width, y: currentNode.y * map.height }
         : { x: 0, y: 0 };

     // Villain trail: use the first node as a stub position until E-11 provides real data
     const villainNode = map.nodes[0];
     const villainPixelPosition =
       villainNode !== undefined
         ? { x: villainNode.x * map.width, y: villainNode.y * map.height }
         : null;

     return (
       <div
         className="relative overflow-hidden"
         style={{ width: map.width, height: map.height }}
       >
         <MapCanvas
           onNodeSelect={handleNodeSelect}
           currentNodeId={currentNodeId}
           animateTo={animateTo}
           onAnimationEnd={handleAnimationEnd}
         />
         <PlayerMarker position={currentPixelPosition} sprite={course.player.sprite} />
         {villainPixelPosition !== null && (
           <VillainTrail position={villainPixelPosition} sprite={course.villain.trailSprite} />
         )}
       </div>
     );
   }
   ```

2. The `position: relative` wrapper here and in `MapCanvas` may be redundant — if `MapCanvas` already provides the containing block, `MapScreen` can omit its own wrapper and render children inside `MapCanvas` instead. Choose whichever avoids double-wrapping at the same dimensions. The implementation detail is up to the developer; the acceptance criterion is that all children are correctly positioned relative to the same origin.

3. The villain trail stub (using `map.nodes[0]`) is clearly wrong at runtime but satisfies TypeScript and produces a visible marker for development. Comment it clearly. Replace during E-11 wiring.

4. Wire `MapScreen` into `App.tsx` as the default view:
   ```tsx
   // In App.tsx
   import { MapScreen } from '@/components/Map/MapScreen';

   export function App(): React.ReactElement {
     // TODO(E-11): switch on lesson active state
     return <MapScreen />;
   }
   ```
