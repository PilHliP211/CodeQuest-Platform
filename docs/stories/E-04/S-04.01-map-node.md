## What

`MapNode.tsx` renders a single lesson node on the world map. It has three visual states driven by a `state` prop:

- **`locked`** ‚Äî greyed out, no pointer cursor, not interactive. The player has not yet unlocked this node.
- **`available`** ‚Äî highlighted, pointer cursor, clickable. The player can start this lesson.
- **`completed`** ‚Äî normal color with a flag icon overlay in the top-right corner. The player has finished all three phases.

The component does not compute its own state ‚Äî it receives `state` as a prop. The caller (`MapCanvas`) derives state from the progress store.

## Tasks

1. Create `src/components/Map/MapNode.tsx`:
   ```tsx
   import type { MapNode as MapNodeData } from '@/types/content';

   type NodeState = 'locked' | 'available' | 'completed';

   interface MapNodeProps {
     node: MapNodeData;
     state: NodeState;
     onClick?: () => void;
   }

   export function MapNode({ node, state, onClick }: MapNodeProps): React.ReactElement {
     const isInteractive = state === 'available';

     return (
       <button
         type="button"
         className={[
           'absolute flex items-center justify-center',
           'w-10 h-10 rounded-full border-2 font-pixel text-xs',
           'translate-x-[-50%] translate-y-[-50%]',
           state === 'locked' && 'opacity-40 cursor-not-allowed border-gray-500 bg-gray-700 text-gray-400',
           state === 'available' && 'cursor-pointer border-yellow-400 bg-yellow-900 text-yellow-200 ring-2 ring-yellow-400 ring-offset-1 hover:scale-110 transition-transform',
           state === 'completed' && 'cursor-default border-green-400 bg-green-900 text-green-200',
         ]
           .filter(Boolean)
           .join(' ')}
         style={{ left: node.x * 100 + '%', top: node.y * 100 + '%' }}
         disabled={!isInteractive}
         onClick={isInteractive ? onClick : undefined}
         aria-label={`${node.label} ‚Äî ${state}`}
         aria-disabled={!isInteractive}
       >
         {state === 'completed' && (
           <span className="absolute -top-1 -right-1 text-sm" aria-hidden="true">
             üö©
           </span>
         )}
         <span className="truncate px-1">{node.label}</span>
       </button>
     );
   }
   ```

2. The `style` positions the node using the content pack's 0-1 normalized `x`/`y` coordinates as percentages. This works because the parent container (`MapCanvas`) is `position: relative` with explicit pixel dimensions. The `translate-x/y-[-50%]` Tailwind classes center the button on the coordinate point.

3. The `disabled` attribute on `<button>` handles pointer events for `locked` and `completed` states. Do not use `pointer-events: none` via inline style ‚Äî it breaks keyboard navigation in unexpected ways.

4. Do not use `any` for the `node` prop. `MapNodeData` must be imported from `src/types/content.ts` (created in E-03).
