## What

The world map shows each lesson node as `locked`, `available`, or `completed`. These states are derived at render time from the progress store and the content pack's node graph. This story creates `mapStateDeriver.ts` and wires it into `MapCanvas`.

A node is `completed` if its lesson has `phase3Complete: true`. A node is `available` if it's `initiallyUnlocked` OR if all nodes whose `unlocksOnComplete` array includes this node's ID are completed. Everything else is `locked`.

## Tasks

1. Create `src/engine/mapStateDeriver.ts`:
   ```ts
   import type { MapNode } from '@/types/content';
   import { getLessonProgress } from './progressStore';

   export type NodeState = 'locked' | 'available' | 'completed';

   export function deriveNodeStates(
     nodes: MapNode[],
     packId: string,
   ): Record<string, NodeState> {
     const result: Record<string, NodeState> = {};

     // First pass: mark completed nodes
     for (const node of nodes) {
       const progress = getLessonProgress(packId, node.id);
       if (progress?.phase3Complete === true) {
         result[node.id] = 'completed';
       }
     }

     // Second pass: mark available and locked
     for (const node of nodes) {
       if (result[node.id] === 'completed') continue;

       if (node.initiallyUnlocked) {
         result[node.id] = 'available';
         continue;
       }

       // A node is available if any other node's unlocksOnComplete contains this node's ID
       const isUnlocked = nodes.some(
         (other) =>
           result[other.id] === 'completed' &&
           other.unlocksOnComplete?.includes(node.id) === true,
       );

       result[node.id] = isUnlocked ? 'available' : 'locked';
     }

     return result;
   }
   ```

2. Update `MapCanvas.tsx` (E-04, S-04.05) to call `deriveNodeStates` instead of the stub that returns `'available'` for all nodes:
   ```tsx
   const { course } = useContent();
   const nodeStates = deriveNodeStates(course.map.nodes, course.id);
   ```

3. `deriveNodeStates` reads from `localStorage` on every render — this is acceptable for v1 (no hot render loop). If performance is a concern, memoize with `useMemo` in `MapCanvas`.
