## What

The orchestrating component for the entire narrative system. It receives a `NarrativeScene[]` (a full `NarrativeScript`), tracks the current scene index, and renders the active `SceneRenderer`. When the last scene is complete, it calls `onComplete()`. It also renders the skip button (S-05.06).

`NarrativePlayer` is the component that E-11 (Lesson Runner) will mount and unmount at phase boundaries. Its interface should stay stable — E-11 should only need to pass in `script` and `onComplete`.

## Tasks

1. Create `src/components/Narrative/NarrativePlayer.tsx`:
   ```tsx
   import { useState } from 'react';
   import type { NarrativeScene } from '@/types/content';
   import { SceneRenderer } from './SceneRenderer';

   export interface NarrativePlayerProps {
     script: NarrativeScene[];
     onComplete: () => void;
     skippable?: boolean;
   }

   export function NarrativePlayer({
     script,
     onComplete,
     skippable = true,
   }: NarrativePlayerProps): React.ReactElement {
     const [sceneIndex, setSceneIndex] = useState(0);

     const handleSceneComplete = (): void => {
       if (sceneIndex < script.length - 1) {
         setSceneIndex((i) => i + 1);
       } else {
         onComplete();
       }
     };

     if (script.length === 0) {
       onComplete();
       return <></>;
     }

     return (
       <div className="relative w-full h-full">
         {skippable && (
           <button
             className="absolute top-3 right-3 font-pixel text-xs text-white/70 hover:text-white z-10"
             aria-label="Skip narrative"
             onClick={onComplete}
           >
             Skip ▶▶
           </button>
         )}
         <SceneRenderer
           key={sceneIndex}
           scene={script[sceneIndex]}
           onSceneComplete={handleSceneComplete}
         />
       </div>
     );
   }
   ```
2. Using `key={sceneIndex}` on `SceneRenderer` automatically resets its internal `lineIndex` state when the scene changes — this is the cleanest approach and avoids a `useEffect` reset pattern.
3. The empty-script guard (`script.length === 0`) must not call `onComplete()` during render. Move it into a `useEffect` with `[]` dependency:
   ```tsx
   useEffect(() => {
     if (script.length === 0) onComplete();
   }, []); // eslint-disable-line react-hooks/exhaustive-deps — intentionally fires once
   ```
   Wait — the project bans `eslint-disable`. Instead, pass `onComplete` as a ref so it doesn't appear in deps:
   ```tsx
   const onCompleteRef = useRef(onComplete);
   useEffect(() => {
     if (script.length === 0) onCompleteRef.current();
   }, []);
   ```
4. Export a barrel `src/components/Narrative/index.ts` that re-exports `NarrativePlayer` so callers can import from `@/components/Narrative`.
