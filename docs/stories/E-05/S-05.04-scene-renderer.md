## What

Renders a single `NarrativeScene`. It displays the scene's optional background image, the current dialogue line's portrait (via `PortraitDisplay`), and the current dialogue line's text (via `DialogueBox`).

`SceneRenderer` owns the current line index with `useState`. Each call to `onAdvance` from `DialogueBox` increments the index. When the last line has been advanced past, `SceneRenderer` calls the parent's `onSceneComplete()` callback instead of incrementing further.

The background image, if present, fills the upper portion of the scene area above the dialogue box.

## Tasks

1. Create `src/components/Narrative/SceneRenderer.tsx`:
   ```tsx
   import { useState } from 'react';
   import type { NarrativeScene } from '@/types/content';
   import { DialogueBox } from './DialogueBox';
   import { PortraitDisplay } from './PortraitDisplay';

   export interface SceneRendererProps {
     scene: NarrativeScene;
     onSceneComplete: () => void;
     autoAdvanceMs?: number;
   }

   export function SceneRenderer({
     scene,
     onSceneComplete,
     autoAdvanceMs,
   }: SceneRendererProps): React.ReactElement {
     const [lineIndex, setLineIndex] = useState(0);
     const currentLine = scene.dialogue[lineIndex];

     const handleAdvance = (): void => {
       if (lineIndex < scene.dialogue.length - 1) {
         setLineIndex((i) => i + 1);
       } else {
         onSceneComplete();
       }
     };

     return (
       <div className="relative w-full h-full flex flex-col">
         {/* Background */}
         {scene.background != null && (
           <div
             className="flex-1 bg-cover bg-center"
             style={{ backgroundImage: `url(${scene.background})` }}
             aria-hidden="true"
           />
         )}

         {/* Portrait + Dialogue row */}
         <div className="flex items-end">
           <PortraitDisplay
             portrait={currentLine.portrait}
             alt={`${currentLine.speaker} portrait`}
           />
           <DialogueBox
             speaker={currentLine.speaker}
             text={currentLine.text}
             onAdvance={handleAdvance}
           />
         </div>
       </div>
     );
   }
   ```
2. Reset `lineIndex` to `0` when the `scene` prop changes (use a `key` on the component at the call site in `NarrativePlayer`, or a `useEffect` that resets on scene identity change).
3. The `autoAdvanceMs` prop wiring is handled in S-05.07 — leave the prop in the interface now so S-05.07 is a small addition rather than a refactor.
4. Guard against an empty `dialogue` array: if `scene.dialogue.length === 0`, call `onSceneComplete()` immediately on mount via `useEffect`.
