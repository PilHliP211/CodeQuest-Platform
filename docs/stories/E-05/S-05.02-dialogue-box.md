## What

The visual dialogue box displayed at the bottom of the screen during narrative scenes. It shows the speaker's name in a header band and the dialogue text below. A blinking "tap to continue" arrow indicator appears when the player can advance. Clicking or pressing Enter/Space calls `onAdvance`. The entire component is styled with a pixel-art border using Tailwind utility classes.

The `DialogueBox` does not know about scenes or lines — it simply renders whatever `speaker` and `text` it is given. State management (which line is current) belongs in `SceneRenderer` (S-05.04).

## Tasks

1. Create `src/components/Narrative/DialogueBox.tsx`:
   ```tsx
   export interface DialogueBoxProps {
     speaker: string;
     text: string;
     onAdvance: () => void;
   }

   export function DialogueBox({ speaker, text, onAdvance }: DialogueBoxProps): React.ReactElement {
     // ...
   }
   ```
2. Layout (Tailwind):
   - Outer container: fixed bottom strip, full width, `bg-gray-900 border-t-4 border-white` (pixel-art border). Minimum height of ~6rem so short lines don't collapse.
   - Speaker name band: `font-pixel text-yellow-300 text-xs px-4 pt-3 pb-1` — uses `font-pixel` Tailwind alias for Press Start 2P.
   - Text area: `font-pixel text-white text-xs leading-relaxed px-4 pb-4`. Allow text to wrap naturally.
   - Advance indicator: a `▼` or `►` character positioned bottom-right, animated with a CSS `animate-bounce` or a custom `@keyframes blink` added via Tailwind `animate-[blink_1s_step-end_infinite]`.
3. Clicking anywhere on the dialogue box, or pressing Enter/Space while it is focused, calls `onAdvance`:
   ```tsx
   <div
     role="button"
     tabIndex={0}
     aria-label="Advance dialogue"
     onClick={onAdvance}
     onKeyDown={(e) => {
       if (e.key === 'Enter' || e.key === ' ') {
         e.preventDefault();
         onAdvance();
       }
     }}
   >
   ```
4. The text area should use `aria-live="polite"` so screen readers announce each new line.
5. No internal state — this is a pure presentational component.
