# S-01.07: Install ESLint

**Epic:** E-01
**Size:** XS
**Dependencies:** S-01.01

## Skills to Load

- `dependency-policy` — to confirm these packages are on the approved list

## What

Install ESLint and the TypeScript ESLint packages. No rules configured yet — that's S-01.08.

## Tasks

1. Install:
   ```
   npm install -D eslint @eslint/js typescript-eslint eslint-plugin-react-hooks eslint-plugin-react-refresh
   ```
2. Verify `npx eslint --version` works
3. Add `lint` and `lint:fix` scripts to `package.json`:
   ```json
   "lint": "eslint src/",
   "lint:fix": "eslint src/ --fix"
   ```

## Done When

- [ ] All five ESLint packages installed as dev dependencies
- [ ] `lint` and `lint:fix` scripts present in `package.json`
- [ ] `npx eslint --version` reports a v9+ version
