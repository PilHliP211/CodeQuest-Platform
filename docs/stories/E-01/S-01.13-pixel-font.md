# S-01.13: Press Start 2P Pixel Font

**Epic:** E-01
**Size:** XS
**Dependencies:** S-01.12

## Skills to Load

None required.

## What

Add the Press Start 2P pixel-art font from Google Fonts and expose it as a Tailwind utility (`font-pixel`).

## Tasks

1. Add the Google Fonts `<link>` to `index.html` `<head>`:
   ```html
   <link rel="preconnect" href="https://fonts.googleapis.com">
   <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
   <link href="https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap" rel="stylesheet">
   ```
2. Extend the Tailwind theme in `src/index.css`:
   ```css
   @import 'tailwindcss';

   @theme {
     --font-pixel: 'Press Start 2P', cursive;
   }
   ```
3. Apply `font-pixel` to the placeholder heading in `App.tsx` to verify

## Done When

- [x] Font loads in the dev server (visible in DevTools Network tab)
- [x] `className="font-pixel"` renders Press Start 2P
- [x] Build succeeds with the font reference
