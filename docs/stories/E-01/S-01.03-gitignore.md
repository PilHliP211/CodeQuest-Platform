# S-01.03: .gitignore

**Epic:** E-01
**Size:** XS
**Dependencies:** S-01.01

## Skills to Load

None required.

## What

Add a `.gitignore` that prevents committing build output, node_modules, and local environment files.

## Tasks

1. Create `.gitignore`:
   ```
   node_modules/
   dist/
   *.local
   .env
   .env.*
   .DS_Store
   coverage/
   .vite/
   ```
2. Confirm `git status` no longer shows `node_modules/` or `dist/`

## Done When

- [x] `.gitignore` exists at repo root
- [x] `node_modules/` is not tracked
- [x] `dist/` is not tracked
- [x] Local `.env*` files are ignored
