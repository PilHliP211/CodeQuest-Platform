# S-01.10: Install Prettier

**Epic:** E-01
**Size:** XS
**Dependencies:** S-01.01

## Skills to Load

- `formatting` — the canonical Prettier config and rules of engagement
- `dependency-policy` — to confirm Prettier is approved

## What

Install Prettier and create the config file. ESLint integration comes in S-01.11.

## Tasks

1. Install: `npm install -D prettier`
2. Create `.prettierrc`:
   ```json
   {
     "semi": true,
     "singleQuote": true,
     "trailingComma": "all",
     "printWidth": 100,
     "tabWidth": 2,
     "arrowParens": "always",
     "endOfLine": "lf"
   }
   ```
3. Create `.prettierignore`:
   ```
   dist/
   node_modules/
   *.md
   coverage/
   ```
4. Add scripts to `package.json`:
   ```json
   "format": "prettier --write \"src/**/*.{ts,tsx,json,css}\"",
   "format:check": "prettier --check \"src/**/*.{ts,tsx,json,css}\""
   ```
5. Run `npm run format` once to normalize the scaffold
6. Run `npm run format:check` to confirm clean state

## Done When

- [ ] Prettier installed
- [ ] `.prettierrc` and `.prettierignore` exist
- [ ] `format` and `format:check` scripts present
- [ ] `npm run format:check` exits 0
