# S-01.05: Folder Structure

**Epic:** E-01
**Size:** XS
**Dependencies:** S-01.01

## Skills to Load

- `architecture-overview` — the directory layout this story creates
- `naming-conventions` — for the casing of subdirectories
- `anti-patterns` — to remember not to create barrel `index.ts` files

## What

Create the directory structure from `docs/ARCHITECTURE.md`. Use `.gitkeep` to track empty directories.

## Tasks

1. Create directories under `src/`:
   ```
   src/
   ├── engine/
   │   └── interpreter/
   ├── components/
   │   ├── Profile/
   │   ├── Map/
   │   ├── HUD/
   │   ├── Narrative/
   │   └── Collection/
   ├── editor/
   ├── renderer/
   └── types/
   ```
2. Add a `.gitkeep` to each empty directory
3. Do NOT create any barrel `index.ts` files — they get added as directories grow real exports

## Done When

- [ ] All directories from ARCHITECTURE.md exist
- [ ] All directories are tracked by git
- [ ] No premature `index.ts` barrel files
