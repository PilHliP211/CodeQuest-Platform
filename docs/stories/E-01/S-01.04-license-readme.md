# S-01.04: LICENSE and README

**Epic:** E-01
**Size:** XS
**Dependencies:** S-01.01

## Skills to Load

None required.

## What

Add a license file and a minimal README. The README is intentionally short — the planning docs in `docs/` cover everything else.

## Tasks

1. Create `LICENSE` (MIT)
2. Create or update root `README.md` with:
   - Project name and one-line description
   - The five `npm run` commands
   - Link to `docs/` for planning documents
   - No badges, no marketing copy

## Done When

- [ ] `LICENSE` exists with MIT text
- [ ] `README.md` is concise (<40 lines) and includes commands + a docs link
- [ ] No empty sections or placeholder TODOs
