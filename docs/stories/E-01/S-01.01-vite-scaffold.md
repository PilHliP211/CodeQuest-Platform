# S-01.01: Vite + React + TypeScript Scaffold

**Epic:** E-01
**Size:** XS
**Dependencies:** none

## Skills to Load

- `architecture-overview` — to know what file structure you're heading toward
- `anti-patterns` — to avoid bad defaults from the Vite template

## What

Initialize a Vite project with the React + TypeScript template. Verify the dev server runs and the production build works.

## Tasks

1. Run `npm create vite@latest . -- --template react-ts` (in-place; the repo already exists)
2. Run `npm install`
3. Replace `App.tsx` body with a minimal "CodeQuest" heading — remove all Vite boilerplate
4. Delete `App.css` and the body of `index.css` (will be replaced by Tailwind in S-01.12)
5. Delete the Vite logo asset and any other boilerplate files
6. Verify `npm run dev` starts cleanly
7. Verify `npm run build` produces a `dist/` directory

## Done When

- [ ] `npm run dev` starts the dev server with no errors
- [ ] `npm run build` produces a working static build in `dist/`
- [ ] `App.tsx` renders a minimal placeholder ("CodeQuest" heading or similar)
- [ ] No Vite boilerplate files remain (`App.css`, logo SVG, etc.)
- [ ] `package.json` has `dev`, `build`, `preview` scripts from the template
