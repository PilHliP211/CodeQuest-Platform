# S-01.12: Install Tailwind CSS

**Epic:** E-01
**Size:** XS
**Dependencies:** S-01.01

## Skills to Load

- `dependency-policy` — Tailwind is approved; don't substitute another CSS framework

## What

Install Tailwind CSS and wire it into Vite. This story is just installation and the smallest possible verification — theme customization is in later stories.

## Tasks

1. Install: `npm install -D tailwindcss @tailwindcss/vite`
2. Add the Tailwind plugin to `vite.config.ts`:
   ```ts
   import tailwindcss from '@tailwindcss/vite';

   export default defineConfig({
     plugins: [react(), tailwindcss()],
   });
   ```
3. Replace `src/index.css` contents with:
   ```css
   @import 'tailwindcss';
   ```
4. Add a Tailwind utility class to `App.tsx` (e.g., `<h1 className="text-3xl font-bold">CodeQuest</h1>`)
5. Run `npm run dev` and confirm the heading is styled

## Done When

- [x] Tailwind packages installed
- [x] Vite plugin configured
- [x] `index.css` imports Tailwind
- [x] Tailwind utility classes apply in the dev server
- [x] `npm run build` produces a build with purged CSS
