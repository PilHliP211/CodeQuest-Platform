# S-01.16: Install Husky

**Epic:** E-01
**Size:** XS
**Dependencies:** S-01.01

## Skills to Load

- `dependency-policy` — both packages are approved
- `ci-and-quality-gates` — pre-commit hook is the local-side gate

## What

Install Husky and initialize the Git hooks directory. Empty hook for now — `lint-staged` config comes in S-01.17.

## Tasks

1. Install: `npm install -D husky lint-staged`
2. Run `npx husky init` (creates `.husky/` and adds a `prepare` script to `package.json`)
3. Confirm `.husky/pre-commit` exists (default contents are fine for now)

## Done When

- [ ] `husky` and `lint-staged` installed
- [ ] `.husky/pre-commit` exists
- [ ] `package.json` has `"prepare": "husky"` script
- [ ] `npm install` runs the prepare script automatically
