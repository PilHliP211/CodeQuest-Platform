# S-01.14: Pixel-Rendering Global CSS

**Epic:** E-01
**Size:** XS
**Dependencies:** S-01.12

## Skills to Load

None required.

## What

Add a global CSS rule so all sprite images and canvas elements render with `image-rendering: pixelated`. This preserves the pixel-art aesthetic at any zoom level.

## Tasks

1. Add to `src/index.css` (after the Tailwind import):
   ```css
   img,
   canvas {
     image-rendering: pixelated;
   }
   ```
2. Verify in the browser: any `<img>` or `<canvas>` element renders with crisp pixels (test with a small upscaled image if available, otherwise just confirm the rule is present)

## Done When

- [ ] Global CSS rule applies `image-rendering: pixelated` to `img` and `canvas`
- [ ] No regressions in `npm run build`
